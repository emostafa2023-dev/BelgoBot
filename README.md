# BelgoBot — Telegram-бот для студентов ИТМО

[![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)](https://www.python.org/)
[![Telegram Bot API](https://img.shields.io/badge/Telegram%20Bot%20API-20.7-blue.svg)](https://core.telegram.org/bots/api)

## 📖 О проекте

**BelgoBot** — это Telegram-бот, разработанный для студентов Университета ИТМО. Он помогает централизовать учебную информацию: расписание, дедлайны и оценки, чтобы студентам не приходилось переключаться между разными сервисами.

## 🚀 Основной функционал

- **Расписание**: просмотр на день/неделю, уведомления о ближайших парах.
- **Дедлайны и задачи**: отслеживание дедлайнов из внешних источников, напоминания, добавление собственных задач.
- **Оценки**: отображение текущих оценок (при доступности данных).
- **Дополнительно**: отправка мемов, анонимные отзывы о преподавателях.

## 🛠 Технологии

- **Язык**: Python 3.10+
- **Библиотека для бота**: `python-telegram-bot`
- **База данных**: SQLite / PostgreSQL
- **Парсинг данных**: HTML (BeautifulSoup) / Google Sheets API
- **Планировщик**: `APScheduler`
